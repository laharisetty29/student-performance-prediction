# Student Performance Prediction (ML)

A machine learning project to predict **student pass/fail outcomes** using academic data stored in Excel.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- OpenPyXL

## ML Algorithms
- Logistic Regression
- Decision Tree

## Dataset
Excel file containing:
- Study hours
- Attendance
- Internal marks
- Assignments
- Previous scores

(Target: `result` → Pass / Fail)

## How to Run
pip install pandas scikit-learn openpyxl
python student_performance.py


## Output
- Model accuracy & classification report
- Excel file with predicted results
